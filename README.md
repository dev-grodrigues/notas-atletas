# <h1 align ="center">Projeto de Certificação DEVStart </h1>
<p align="center" >
Aplicação que recebe o nome e notas de um atleta, calcula a média e exibe o resultado ao usuário
</p>
---

## 🚀 Tecnologias utilizadas
* Node.js
## ▶️ Como executar 
- 1- Clone o repositorio
```bash
git clone https://github.com/dev-grodrigues-notas-atletas
```
- 2- Entre na pasta
```bash
cd notas-atletas
```
- 3- Abra o arquivo 
```bash
node notas-atletas.js
```
